# MEMPHOS
Memphos is an OS written by Me, Mempler.
Memphos will never be anything big since it's just me learning how an OS works under the hood, like really deeeeep.


Requirements:
* Nasm
* GCC
* Make
* Qemu
* qemu-system-i386
* Any Unix Machine

Build Kernel: \
`make kernel`

Run Qemu (Doesn't require `make kernel`): \
`make qemu`
